    This is a stub filesystem purely present to fail meaningfully when someone who explicitly declares fs.s3n.impl=org.apache.hadoop.fs.s3native.NativeS3FileSystem and then tries to create a filesystem off an s3n:// URL. The initialize(URI, Configuration) method will throw an IOException informing the user of their need to migrate.
