TDIGEST.RESET

Syntax
TDIGEST.RESET key
Available in:
Redis CE / Bloom 2.4.0
Time complexity:
O(1)
ACL categories:
@tdigest, @write, @fast,

Resets a t-digest sketch: empty the sketch and re-initializes it.

Required arguments

key is key name for an existing t-digest sketch.

Return value

Simple string reply - OK if executed correctly, or [] otherwise.

Examples

redis> TDIGEST.RESET t
OK
